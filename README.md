# streaming-project

## components

Backend:

- NestJS (NodeJS)
- Prisma (ORM)
- PostgreSQL (Database)
- node-media-server (RTMP and HLS server)
- node-fluent-ffmpeg (Video transcoding API)

Frontend:

- Vite.js
- React
- ChakraUI
- HLS.js (Video playback engine)
- media-chrome (Video player)

Others:

- ffmpeg (Video transcoding)
- OBS (Video streaming via RMTP)
